# 🎮 基础操作指南

在《人类一败涂地》中，小人 Bob 软绵绵的身体拥有两只独立的手臂。你的所有操作最终都会被转换为双臂的力矩输出。本指南将帮助你调试动作，迅速通过前期的肌肉记忆难关。

## ⌨️ 动作映射终端 (键盘操作)

熟记以下基本指令集。每次进入关卡，你都可以直接通过它们来控制 Bob。

*   **前进 / 后退 / 左移 / 右移**：使用 <span class="kbd-key">W</span><span class="kbd-key">A</span><span class="kbd-key">S</span><span class="kbd-key">D</span>。
*   **视角环视**：滑动 <span class="kbd-key">鼠标</span>。控制头部的看天看地，配合抓取。
*   **左手抓取**：按住 <span class="kbd-key mouse-key">鼠标左键</span>，Bob 会伸出左手吸附前方物体。
*   **右手抓取**：按住 <span class="kbd-key mouse-key">鼠标右键</span>，Bob 会伸出右手吸附前方物体。
*   **双脚跳跃**：按下 <span class="kbd-key">Space</span>。起跳的瞬间会带有往前的物理冲力。
*   **极速奔跑**：按住 <span class="kbd-key">Shift</span> 不放，Bob 会弯腰加速奔跑。

## 📐 核心动作原理图解

### 1. 抓取与释放机制
*   **动作逻辑**：只要按住 <span class="kbd-key mouse-key">鼠标左右键</span>，Bob 的双手就会变成高强度吸盘。任何表面（墙壁、铁箱、队友的屁股）只要被触碰就会牢牢黏住。
*   **要领**：松开按键即为释放。如果需要推动重物，建议**双手同时抓取**，然后再按住 <span class="kbd-key">W</span> 往前推或按 <span class="kbd-key">S</span> 往后拉。

### 2. 基础垂直攀爬 (翻墙)
这是游戏中最基础的翻墙技巧，也是获取后续高级邪道的前提：
1.  **助跑靠近**：按住 <span class="kbd-key">Shift</span> + <span class="kbd-key">W</span> 跑向墙面。
2.  **双手举高**：将鼠标视角**向上看**，此时 Bob 会高举双手。
3.  **起跳抓取**：在贴近墙壁的瞬间，按下 <span class="kbd-key">Space</span> 起跳，并同时按住 <span class="kbd-key mouse-key">鼠标左右键</span> 抓住墙沿。
4.  **低头拉起**：抓住墙沿后**不要放手**，将视角慢慢**转向正下方**，同时按住前进 <span class="kbd-key">W</span> 键。此时 Bob 的手臂会收缩，身体会因为物理判定被拉上高台。

### 3. 摆动跨越
*   **应用场景**：当你遇到过宽的深沟，且天花板上垂挂着绳索或铁链时。
*   **解密步骤**：跳起抓牢垂挂物，按住 <span class="kbd-key">W</span> 和 <span class="kbd-key">S</span>，让 Bob 像秋千一样在空中做圆周运动。当摆动到弧线的最高点时，松开鼠标键，身体会因离心力被抛射到对面。

<div class="unlock-tip-box">
  <strong>【解锁技巧：爬墙新手误区】</strong><br>
  很多新手在把身体拉上墙壁时，会因为急于按前进键而松开鼠标。记住：<strong>在双脚平稳站上墙头之前，鼠标左右键绝对不能松开！</strong> 视角转到下方后，直到 Bob 抬腿跨越平台，才算攀爬完成。
</div>
