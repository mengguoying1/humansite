Welcome to the control center! In Human: Fall Flat, Bob's wobbly limbs obey independent physical forces. To unlock advanced glitch movements or speedrun strategies, you must first master the basic control schemas below.

## 🕹️ Control Panel Mapping (Default Inputs)

| Action Category | Action Description | Keyboard Mapping | Controller Mapping |
| :--- | :--- | :--- | :--- |
| **Movement** | Move Bob around | <span class="kbd-key">W</span><span class="kbd-key">A</span><span class="kbd-key">S</span><span class="kbd-key">D</span> | <span class="kbd-key">Left Stick</span> |
| **Camera** | Rotate Bob's head / camera | <span class="kbd-key">Mouse Move</span> | <span class="kbd-key">Right Stick</span> |
| **Left Arm** | Extend left hand to grab | <span class="kbd-key mouse-key">Left Click</span> | <span class="kbd-key">LT</span> / <span class="kbd-key">L2</span> |
| **Right Arm** | Extend right hand to grab | <span class="kbd-key mouse-key">Right Click</span> | <span class="kbd-key">RT</span> / <span class="kbd-key">R2</span> |
| **Jump** | Jump upwards (or jump forward) | <span class="kbd-key">Space</span> | <span class="kbd-key">A Button</span> / <span class="kbd-key">× Button</span> |
| **Run** | Run forward with momentum | <span class="kbd-key">Shift</span> | <span class="kbd-key">LS Click</span> |
| **Play Dead** | Go limp instantly | <span class="kbd-key">T</span> | <span class="kbd-key">B Button</span> / <span class="kbd-key">○ Button</span> |
