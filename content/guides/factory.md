# 🏭 第二关：工厂 (Factory)

> **难度：** ⭐⭐☆☆☆ (新手进阶)
> **目标：** 穿过废弃工厂，到达出口大门
> **核心机制：** 抛掷、推拉、利用传送带

---

## 🎯 关卡概览

工厂是第二关，难度比豪宅稍高。你需要穿过一个废弃的工厂车间，利用传送带、箱子、管道等各种工业设施到达出口。

**关键区域：**
1. 🚪 入口平台 — 起点
2. 📦 传送带区 — 核心解谜区
3. 🪜 高台 & 管道 — 中间区域
4. 🏗️ 大坑 & 吊臂 — 挑战区域
5. 🚪 出口大门 — 终点

---

## 🗺️ 详细攻略

### 阶段一：入口平台

{{< hint info >}}
**起点位置：** 你站在工厂入口的平台上，前方是一个大车间。
{{< /hint >}}

**目标：** 下到车间地面。

**操作：**
1. 走到平台边缘
2. 直接跳下去（不高，不会摔伤）
3. 或者抓住边缘爬下去

**💡 观察：** 注意周围的环境，有很多箱子和传送带。

---

### 阶段二：传送带区 ⭐ 核心解谜

**目标：** 利用传送带和箱子到达对面高台。

**解法一：搭箱子上台（推荐）🟢**
1. 在传送带区域找到 **可移动的箱子**
2. 把箱子拖到高台下方
3. 踩着箱子爬上高台
4. 如果箱子不够高，可以 **叠两个箱子**

**解法二：利用传送带 🟡**
1. 站到传送带上
2. 传送带会把你送到另一端
3. 在传送带末端跳上高台
4. 注意传送带方向，可能需要反向走

**解法三：抛掷箱子 🟡**
1. 抓住一个箱子
2. 跳到半高的平台上
3. 把箱子抛到更高处
4. 踩着抛上去的箱子继续爬

{{< hint tip >}}
**传送带技巧：** 在传送带上逆向走可以"原地踏步"，利用这个特性可以调整位置。
{{< /hint >}}

---

### 阶段三：高台 & 管道

**目标：** 通过管道区域，到达工厂深处。

**步骤：**
1. 从高台跳到管道上
2. 沿着管道走到另一端
3. 注意管道上有 **缝隙**，小心别掉下去
4. 到达管道尽头后跳到下一个平台

**💡 技巧：** 管道上行走时按住 Shift 可以走得更稳。

**捷径：**
- 如果你对自己的操作有信心，可以直接从高台跳到对面的平台上
- 跳过管道区域可以节省时间

---

### 阶段四：大坑 & 吊臂

**目标：** 跨越大坑，到达出口区域。

**解法一：走吊臂 🟢**
1. 找到横跨大坑的 **吊臂/横梁**
2. 双手抓住吊臂
3. 交替移动手臂，**像单杠一样荡过去**
4. 到达另一端后翻上去

**解法二：荡秋千 🟡**
1. 抓住吊臂的一端
2. 摆动身体获得动力
3. 在摆到最高点时松手 + 跳跃
4. 直接飞到对面平台

**解法三：搭桥 🟡**
1. 在周围找到 **长木板或管道**
2. 把木板搭在大坑上形成桥
3. 直接从桥上走过去

{{< hint warning >}}
**注意：** 大坑很深，掉下去要重新爬上来。如果掉下去了，找找坑里有没有其他出路。
{{< /hint >}}

---

### 阶段五：出口大门 🏁

**目标：** 打开大门，通关！

**步骤：**
1. 到达出口大门前
2. 找到大门旁边的 **开关或拉杆**
3. 拉动开关打开大门
4. 走出去 — 通关！🎉

**💡 注意：** 有些版本的大门需要两个人配合才能打开（单人模式可以自己搞定）。

## 🎮 速通技巧

| 技巧 | 说明 | 难度 |
|------|------|:----:|
| **跳过传送带** | 直接从入口跳到高台 | ⭐⭐⭐ |
| **管道捷径** | 跳过管道直接到吊臂区 | ⭐⭐ |
| **超级跳过大坑** | 利用抛掷技巧直接飞过大坑 | ⭐⭐⭐⭐ |
| **邪道路线** | 利用物理bug穿墙到达出口 | ⭐⭐⭐⭐⭐ |

## 🔍 隐藏要素

- **彩蛋：** 工厂角落有一个可以踢的油桶
- **秘密区域：** 传送带下方有一个隐藏的小空间
- **成就相关：** 尝试不踩传送带通关
- **多人玩法：** 一个人站在传送带上，另一个人推箱子过来

## 💡 核心要点总结

1. **箱子是你的好朋友** — 多利用箱子搭路
2. **传送带可逆走** — 利用这个特性调整位置
3. **吊臂要荡起来** — 摆动到最高点再松手
4. **多观察环境** — 工厂里有很多可利用的道具
5. **掉坑别慌** — 找找坑里有没有其他路
